## Common field props

Includes common input field props

## Checkbox group specific props

| prop    | type     | description                           |
|:--------|:---------|:--------------------------------------|
| options | Option[] | See below for Option object structure |

### Option object

| prop     | type     | description         |
|:---------|:---------|:--------------------|
| label    | string   |                     |
| value    | any      |                     |
| disabled | boolean  | Defaults to `false` |
| tooltip  | string or TextNode |          |
