## Common field props

Includes common input field props

## Select specific props

| prop    | type     | description                           |
|:--------|:---------|:--------------------------------------|
| width | string? | 'xxs', 'xs', 's', 'm', 'l', 'xl' |
| options | Option[] | See below for Option object structure |

### Option object

| prop  | type   | description               |
|:------|:-------|:--------------------------|
| label | string |                           |
| value | any    |                           |
| disabled | boolean    | Defaults to `false`                          |
